# Glossary

The repository makes use of [Contextive](https://contextive.tech) to maintain
smart glossaries. Contextive provides features to integrate glossaries seamless
into the daily work. For example, with direct integration into code editors.
Check out the related [tool
documentation](./documentation/tool-usage/contextive.md) how to use it. This
also works outside of the repository for users of the XJustiz-Converter library.

Glossaries are fined in `*.glossary.yaml` files and can be discovered in the
repository. They can be read as plain text without specialized tooling.
